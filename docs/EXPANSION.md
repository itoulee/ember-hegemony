# 后续扩展建议（含 R18 / 成熟向分轨）

在线试玩：https://itoulee.github.io/ember-hegemony/  
（请使用**带末尾斜杠**的地址。）

---

## 1. 产品分轨（强烈建议）

| 轨道 | 内容 | 发布渠道 |
|------|------|----------|
| **主线（全年龄/成熟 16+）** | 战争、权谋、联姻政治、监禁/政变（不写性描写） | GitHub Pages / Steam 全年龄 |
| **R18 扩展包（可选）** | 成人向关系、后宫/誓约事件、俘虏调教等（仅成人） | 独立仓库/付费 DLC / 模组，**不并入默认 Pages** |

雷神系乐趣很大一部分来自「权谋 + 关系自由度」，但 **默认开源网页版应保持可公开传播**。R18 用 **内容包 + 年龄门** 加载，而不是写死进主包。

---

## 2. R18 若要做：推荐架构（不要直接塞进主包）

### 2.1 技术结构

```
web/                 # 全年龄主程序
mods/
  mature-r18/        # 可选，默认不部署到 Pages
    mod.json         # { "id":"r18", "requiresAge":18, "events":[...] }
    events/*.json
    characters/*.json
```

- 启动时若检测到 R18 模组 → **年龄确认门**（仅本地记忆，无法真实验证，但尽到产品责任）
- 事件字段增加：`rating: "all" | "mature" | "r18"`
- 主程序过滤：未启用 R18 时丢弃 `rating=r18` 事件
- **禁止**：任何未成年描写、禁止乱伦若目标平台不允许；原作雷神的极端线建议**不要复刻**

### 2.2 玩法层（对标雷神、可发行的做法）

| 系统 | 全年龄主包 | R18 包增强 |
|------|------------|------------|
| 联姻 | 条约、关系、同盟 | 誓约等级、同居、后宫管理 UI |
| 人物好感 | 馈赠/宴饮 | 约会、亲密事件（可淡出黑屏） |
| 俘虏 | 交换、招降 | 审讯/调教线（仅成人包） |
| 结局 | 制霸/政变/建国 | 角色 HE/BE 多结局图鉴 |

### 2.3 内容生产

- 事件仍用 JSON，文案外包或社区贡献
- 立绘/CG 资源单独授权，MIT 代码 + 资源 CC/专有拆分
- 审核表：分级、敏感词、平台 ToS（Steam/itch 成人区）

### 2.4 本仓库已落地的接口骨架

| 项 | 路径 / 操作 |
|----|-------------|
| 年龄门 | 游戏内「R18 内容包」开关 / 加载接口包时弹出 |
| 分级过滤 | 事件字段 `rating: "r18"`，未启用时不进入抽卡池 |
| 占位包 | `web/mods/mature-r18/mod.json`（weight=0，无露骨正文） |
| 关系钩子 | `relationships.levels` 含 `intimate`，仅启用 R18 后由包声明 |

**主仓库与 Pages 仍不包含露骨性内容。**  
真正成人正文请放独立私有包。

## 3. 非 R18：高价值扩展（建议优先）

1. **目标罗盘 / 胜利条件 UI**（制霸进度条）  
2. **更多手写角色线**（每势力 1 条 5 事件链）  
3. **存档云导出**（JSON 文件下载/导入）  
4. **Godot 对齐 P1/P2**（桌面包）  
5. **真正战棋编辑器**（笔刷地形、放置单位）  
6. **Steam 页 + itch.io 镜像**（国内访问备用）  
7. **教程可跳过 + 第一次提示气泡**  
8. **移动端按钮加大与虚拟摇杆式星图**

## 4. 与原作雷神的「像」与「不像」

| 像 | 不像（有意） |
|----|----------------|
| 身份跃迁、权谋、星域制霸 | 默认无 H 内容 |
| 失败可转生 | 极端伦理线不复刻 |
| 事件表驱动 | UI 现代化、可网页玩 |

---

## 5. 合规底线（做 R18 前必读）

- 仅 18+ 角色；内容包与主包分离  
- 年龄门 + 明确分级标签  
- 不写真实犯罪教唆；虚构权谋暴力按分级标注  
- 开源协议与美术资源授权分开写清
