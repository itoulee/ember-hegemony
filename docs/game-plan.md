# 《星域制霸：余烬》（暂定名）详细游戏计划

> 定位：一款**类似《雷神天制霸7 PUK》核心体验**的现代回合制宇宙策略 SLG。  
> 继承：多身份、星域制霸、内政科技、派系政变、失败可转生。  
> 不继承：成人向与违法极端内容；改为成熟向（可含暴力/权谋）可分级发行版本。  
> 关联分析：`docs/superpowers/specs/2026-07-18-raizin7-analysis.md`

### 已冻结产品决策（2026-07-18）

| 决策项 | 结论 |
|--------|------|
| 引擎 | **Godot 4** |
| 题材调性 | **硬核科幻**（ thruster 物理感、资源/信息约束、少魔法黑箱） |
| 政治联姻 | **需要**（全年龄：条约/继承/好感，无成人内容） |
| 语言 | **简体中文**（UI/文案/文档；代码标识符可用英文） |
| 商业模式 | **开源**（推荐 MIT 或 GPL-3.0，仓库公开；非 Steam 商业首发） |
| 战斗 | **v1 快速战报** + **架构预留六角战棋**（接口与数据与战报共用） |

## 0. 产品一句话

**在一张活的星域地图上，从任何人开始，用舰队、科技、外交与权谋，把自己的故事写成「制霸」或无数条同样成立的歧路。**

**正式工作名**：**星域制霸：余烬**（下文称 **《余烬》**）。  
调性锚点：硬科幻重建期政治—— thruster 航运、密钥协议、正统性来自信息与舰队，而非玄学天命。

## 1. 设计目标与非目标

### 1.1 目标（Must）

1. **30 分钟内**理解「我是谁、这回合能干什么、怎样变强」。
2. **一局 8–20 小时**可通关标准星域；速通剧本 3–5 小时。
3. **至少 4 种开局身份**，且局内可迁移。
4. **国家层 + 个人层**双循环同时成立。
5. **科技代差**可感知；落后有追赶手段（间谍、盟研、俘获蓝图）。
6. **失败不等于删号**：入狱、流亡、降身份、换势力再起。
7. **无成人内容依赖**；情感线为可选的「誓约/联姻/挚友」政治关系。

### 1.2 非目标（Won’t in v1）

- 完整 4X 行星表面经营（建造到每块地）
- 3A 级实时太空战斗
- MMO / 强联网（v1 单机；预留模组与日后异步）
- 上百名全语音全立绘角色（v1 用「核心角色 + 程序化武将」）

### 1.3 成功指标（设计验收）

| 指标 | 标准 |
|------|------|
| 身份迁移 | 任意开局 50 回合内至少有 1 条合法迁身份路径 |
| 政变 | 仕官路线可在不读档下完成一次政变（难度正常） |
| 制霸 | 霸王开局可在标准地图统一 |
| 海盗建国 | 海盗可占 ≥1 中立星并建国 |
| 教学 | 新玩家无攻略过完「第一章教程剧本」 |

## 2. 体验支柱（三大乐趣）

1. **权力跃迁**：浪人 → 军官 → 党魁 → 君王  
2. **星域棋感**：同盟、背刺、代差、集火  
3. **人物戏剧**：忠诚、仇敌、继承人、丑闻、赦免  

每条系统功能必须服务其中至少一条，否则砍。

## 3. 世界观设定（原创，避免直接抄IP）

### 3.1 背景（硬科幻口径）

「大断裂」：一次**协调失败的网络级级联**摧毁了多数超光速中继。残存文明依赖：

- **有限 FTL 走廊**（可测绘、可封锁、有带宽/冷却，不是随意瞬移）
- **正交石（Ortholite）**：高能同位素/超导体原料，用于中继维护与舰用反应堆（战略资源，有储量与运力）
- **航道密钥**：密码学+硬件令牌，决定谁有权使用高带宽走廊（可窃取、可作废、可联姻联署）

各**航阀**占据节点。旧帝国法统只剩协议残骸；**制霸**= 控制主环中继 + 在密钥议会获得法定多数。

文案避免：魔法、神谕、无解释超能力。可以有：失效 AI 遗迹、基因改造阶层、信息战、辐射带、长延迟通信。

### 3.2 地理

- **星域图**：节点 = 恒星系；边 = 航道（可封锁、收税、断盟后禁行）
- **系内**：1 主行星 + 0–3 卫星/殖民地（简化经济）
- **特殊点**：中立港、海盗巢、遗迹、科研站、监狱星

### 3.3 势力类型

| 类型 | 行为倾向 |
|------|----------|
| 军国航阀 | 扩张、高军费 |
| 商盟 | 贸易、雇佣 |
| 教团/法统残余 | 意识形态、继承法 |
| 海盗邦联 | 劫掠、可被雇佣 |
| 公司国 | 科技快、兵弱 |

## 4. 身份系统（核心）

### 4.1 四种开局身份

| 身份 | 起始资源 | 主循环 | 特殊胜利/成就 |
|------|----------|--------|----------------|
| **执政官**（霸王） | 1 势力、数星、舰队、内阁 | 内政+战争+防政变 | 制霸；盛世立法 |
| **航官**（仕官） | 品阶、工资、任务 | 功勋→升职→派系→政变/辅政 | 黄袍；权臣结局 |
| **流民**（平民） | 自由移动、事件 | 游历、技能、机遇入仕/入盗 | 布衣卿相；隐士 |
| **掠航者**（海盗） | 快船、恶名、黑市 | 劫掠、勒索、占星建国 | 黑旗帝国 |

### 4.2 身份迁移规则（v1）

```
流民 --(功绩/举荐/救主事件)--> 航官
航官 --(政变成功 / 合法继承)--> 执政官
航官 --(叛逃/免职)--> 流民 或 掠航者
掠航者 --(占星建国并获承认)--> 执政官
执政官 --(亡国流亡)--> 流民 或 掠航者
任意 --(重罪审判)--> 囚徒状态（有限行动）--> 越狱回原身份或流民
```

**囚徒**不是独立开局，是状态：每回合 1 次「贿赂 / 越狱 / 服刑减期 / 对外联络」。

### 4.3 品阶与权限（航官）

| 品阶带 | 示例权限 |
|--------|----------|
| 初阶 | 接 E/D 任务、访问同阶、训练 |
| 中阶 | 带小舰队、外交随员、建小派系 |
| 高阶 | 方面军、提案立法、弹劾程序 |
| 极阶 | 摄政、合法逼宫条件解锁 |

权限用**布尔能力旗标**实现，避免「有官没权」的空数值。

## 5. 回合与行动

### 5.1 时间

- 1 回合 = 1 **星月**
- 每角色默认 **2 行动点（AP）**；执政官额外 +1「国令」AP（仅内政/外交/任命）

### 5.2 行动表（摘要）

**个人类**：移动、访问、训练、使用道具、接任务、进入黑市、休息（清伤）  
**军事类**：编队、出击、围城、护航、撤退  
**内政类**（需权限）：任命、税收政策、研发、肃清、特赦、建设重点  
**外交类**：结盟、从属、索贡、破盟、交换俘虏、雇佣掠航者  
**权谋类**：拉票、揭丑闻、策反、部署保镖、启动政变倒计时  

### 5.3 结束回合

- 有未用 AP 结束：获得 **勤务点**（用于兑换训练书/小道具，**保底表**，禁止纯靠 S/L）
- 世界 AI：全部势力与重要 NPC 结算

### 5.4 伤病

- 战斗/事件可导致 **伤病计数器**（N 回合行动受限）
- 医院星/名医道具可缩短

## 6. 国家层系统

### 6.1 行星

属性：

- `income` 工业/贸易产出  
- `loyalty` 民忠（影响暴动与税收）  
- `defense` 轨道防御等级  
- `dev_slots` 开发槽（有限）  
- `unrest` 动荡  

行动：投资、镇压、宣传、抽丁、建造船坞/研究所/防空。

### 6.2 资源

| 资源 | 用途 |
|------|------|
| 信用点 | 万事货币 |
| 正交石 | 高级舰/跃迁/尖端科技 |
| 影响力 | 外交与议会 |
| 情报点 | 谍报行动 |
| 正统性 | 执政官防政变、外交折扣 |

### 6.3 内阁担当（简化原作）

| 担当 | 职能 | 主属性 |
|------|------|--------|
| 内务 | 忠诚、发掘人才 | 交涉 |
| 军务 | 维修、征兵、编制效率 | 统率 |
| 工务 | 研发速度、合研 | 智力 |
| 卫戍 | 防刺、反谍第一线 | 格技 |
| 外务 | 外交成功率、条约 | 交涉 |
| 巡察 | 反贪、降动荡 | 智力 |

玩家可兼任（惩罚效率）或任命 NPC（忠诚风险）。

### 6.4 科技树（4 分支 × 6 阶）

1. **舰体**：装甲、动力、舰级解锁  
2. **武装**：舰炮、导弹、电子战  
3. **陆战/登舰**：占领速度、镇压力  
4. **政体与后勤**：税收、医疗、情报、行政容量  

规则：

- 每阶研究耗时与正交石递增  
- **同时研究槽**默认 1，工务/建筑可 +1  
- **合研**：同盟双方分摊 50% 成本，共享该节点（条约可规定独占延期）  
- **蓝图俘获**：大胜/奇袭科研站有概率跳过前置的 1 个节点（需平衡）

### 6.5 外交工具箱

- 互不侵犯 / 通航 / 同盟 / 从属  
- 索贡、要人质（人质 = 降低对方毁约，提高己方正统性争议）  
- 破盟：若己方舰队仍在对方领地 → 全军与官员忠诚惩罚  
- **掠航契约**：付费 N 月，每月 1 次可召唤 AI 海盗舰队参战（中前期强）

## 7. 军事系统

### 7.1 战略单位

- **舰队** = 司令官 + 舰位（轻型/巡洋/战列/载机/后勤）  
- 移动消耗 AP 或「航行回合」  
- 补给线：远离船坞维修变慢

### 7.2 战斗分辨率

#### v1：快速战报（默认）

1. 组装 `BattleContext`（双方舰队快照、地形修正、司令、科技）  
2. `IBattleResolver.resolve(context) -> BattleResult`  
3. 战报 UI 播放 3–6 波交火摘要；可选战术节点：集火、防空、撤退、电子压制  
4. 写回伤亡、俘获、士气  

#### 预留：六角战棋（v1 不实装玩法，必留接口）

```
IBattleResolver
├── ReportBattleResolver    # v1 实现
└── HexTacticsBattleResolver  # 后续：载入同 BattleContext，产出同 BattleResult
```

硬性约定（开源协作友好）：

| 约定 | 说明 |
|------|------|
| 唯一结果类型 | `BattleResult`（伤亡、俘获、旗帜、日志条目） |
| 唯一输入 | `BattleContext`（可序列化，便回放/测试） |
| 舰单位数据 | 含 `hull/shield/weapon/mobility` 等战棋可复用字段，战报只读聚合值 |
| 场景入口 | `BattleSession` 可切换 resolver，不改战略层 |
| 地图资源 | `res://battles/hex/` 目录与 `HexGrid` 空模块可先提交 stub |

战略层**禁止**直接改舰血；只许应用 `BattleResult`。

### 7.3 个人决斗

触发：刺杀、政变宫斗、荣誉挑战。  
属性：格技为主，道具与保镖修正。  
执政官死亡 → **继承危机事件链**（指定继承人 / 内战 / 权臣拥立）。

## 8. 人物、派系与权谋

### 8.1 角色属性

- 统率、智力、交涉、格技、魅力  
- 特质标签：贪财、忠义、野心、学者、军阀…  
- 对势力忠诚 vs 对玩家好感（分离，支持「忠于国家但恨你」）

### 8.2 派系

- 高阶航官可建派系，拉拢成员  
- 派系力：在政变/弹劾投票中当票数  
- 暴露后可能被肃清

### 8.3 政变流程（可预期、可防）

1. **筹备**：成员、资金、卫戍渗透 ≥ 阈值  
2. **倒计时** 2–4 回合（可能走漏风声）  
3. **决战回合**：首都控制 + 决斗/巷战 + 舰队站队  
4. **结果**：成功登基 / 流放 / 处死 / 谈判共治  

执政官侧：卫戍担当、正统性、收买、提前逮捕降低阈值。

### 8.4 关系动作（全年龄向）

访问：馈赠、宴饮、议政、策反、赠物、请求协助、结为契盟、**提亲/联姻**。  
**不**做成人选项；亲密关系用「誓约等级」影响事件与忠诚。

### 8.5 政治联姻（已确认需要）

联姻是**外交与继承工具**，不是恋爱养成主轴。

| 要素 | 规则 |
|------|------|
| 发起条件 | 双方（或监护/家主 NPC）同意；品阶/身份限制表驱动；冷却回合 |
| 成本 | 信用点、正交石或领土特许；可能消耗正统性或影响力 |
| 收益 | 同盟加成、通行权、继承声明、忠诚缓冲、联合科研折扣 |
| 联署密钥 | 硬科幻味：婚姻契约可附带**航道密钥联署权**（共管走廊） |
| 破裂 | 离婚/毁约 → 外交惩罚、正统性争议、派系分裂事件 |
| 子嗣（可选抽象） | 用「继承权计数器/继承人标记」表示，不写未成年向内容 |
| UI | 「契约」面板：条款列表 + 接受/修改/拒绝，类似条约 |

与派系/政变交互：联姻对象家族自动倾向己方票仓；敌对联姻可触发「宫廷情报」事件。

## 10. 胜利与终局

### 10.1 主胜利

- **制霸**：控制主航道环 ≥ 80% 节点且正统性 ≥ 阈值，并维持 6 回合  
- **霸约**：通过议会法案成为法定共主（外交/影响力路线）  
- **碎片和平**：多极均势下达成「余烬公约」（和平 tie 向成就）

### 10.2 身份特色结局

- 权臣摄政、布衣卿相、海盗皇帝、流亡复国、隐退航海家  

### 10.3 失败

- 角色死亡且无继承人/无再起事件 → 本周目结束  
- 否则进入流亡/囚徒分支

## 11. 进度与内容范围（分期）

### Phase 0 — 原型（约 4–6 周，单人可压缩）

**目标**：验证「回合 + 身份 + 简单星图 + 一种战斗」好玩。

- 10 节点星图  
- 2 身份：执政官、航官  
- 资源：钱、兵力  
- 无立绘，纯 UI  
- 自动战斗  
- 可占星、可宣战、可升职假数据  

**出口标准**：内部 5 人愿连续玩 1 小时以上。

### Phase 1 — 垂直切片（Vertical Slice）

- 4 身份齐全  
- 内阁 + 科技 2 分支 4 阶  
- 外交：同盟/破盟/从属  
- 政变最小闭环  
- 1 教程 + 1 沙盒  
- 基础原画风格定稿（UI + 3 关键角色）  

### Phase 2 — 内容增厚（可试玩demo）

- 标准图 30–40 节点  
- 科技满树  
- 事件 150+  
- 程序化武将 + 20 名手写关键角色  
- 勤务保底、伤病、间谍  
- 存档、设置、成就  

### Phase 3 — 完整 v1.0

- 5 剧本  
- 平衡与 AI 行为树强化  
- 模组 API（JSON 事件/角色）  
- **全简体中文**（开源文档中文；代码英文标识）  
- 平台：**PC 开源发布**（Git 仓库 + 可导出 Windows 构建；非商业 Steam 首发）  
- 许可证文件与 `CONTRIBUTING.md`（中文）齐全  
- 战棋：`HexTactics` 模块仍为 stub 或实验场景，不阻挡 1.0  

### Phase 4 — 后制（可选）

- 扩展剧本包  
- 深度战术战斗模式  
- 创意工坊

## 12. 技术架构计划

### 12.1 技术栈（已冻结）

| 层 | 选择 | 说明 |
|----|------|------|
| 引擎 | **Godot 4.x** | 2D 星图 + 重 UI；开源友好 |
| 语言 | **GDScript** 为主（热点可后续 C#/C++） | 降低贡献门槛 |
| 数据 | JSON（Godot 原生）+ 可选 CSV 表 | 事件/科技/联姻条款表驱动 |
| 存档 | `Resource` 或 JSON 世界快照 + `save_version` | 含 RNG 种子 |
| UI | Control 控件树 + 自定义列表面板 | 双面板信息密度 |
| 许可证 | 建议 **MIT**（最大兼容）或 **GPL-3.0**（强 copyleft，需你最终选） | 根目录 `LICENSE` |
| 仓库结构 | `game/` 工程、`docs/` 设计、`tools/` 校验脚本 | 中文 README |

### 12.2 核心模块划分

```
App
├── Core (回合循环、RNG 种子、事件总线)
├── World (星图、航道、势力、行星)
├── Actors (角色、属性、关系、身份状态机)
├── Politics (派系、政变、议会、正统)
├── Military (舰队、战斗解析、伤亡)
├── Economy (资源、科技、贸易)
├── Diplomacy (条约、AI 意图)
├── Content (事件脚本、任务、对话)
├── AI (势力 AI、角色 AI)
├── UI (双面板：国家/个人)
└── Meta (存档、设置、成就、模组)
```

### 12.3 确定性

- 同种子同操作可复现（方便测试与回放）  
- 战斗与社交骰子走统一 `RngService`  

### 12.4 AI 分层

1. **势力 AI**：扩张/收缩/结盟欲望效用函数  
2. **角色 AI**：野心值决定是否入派、是否行刺  
3. **战术 AI**：战斗选项权重  

v1 不求完美，但要：**弱者不自杀式宣战、强者会围殴第二名**。
