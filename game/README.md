# Godot 4 桌面原型（C4 核心对齐）

完整功能以 **网页版** 为准：  
https://itoulee.github.io/ember-hegemony/

## 本包已对齐

- 星图移动 / 占星 / 进攻（战报或六角战棋场景）
- 结束回合：收入、势力 AI、事件表
- 政治联姻（最小）
- **存档槽 1–2**（`user://saves/slot_*.json`）
- **目标文案**（状态面板显示控制占比）

## 未完全对齐网页

角色线 HE、后宫、CG 槽、模组工作台、自定义胜利 UI 等仍以网页为主。

## 运行

Godot 4.3+ 打开本目录 `project.godot`，F5。
